### 프로젝 소개
```
카카오톡 플러스친구를 이용해 교내 식당 및 기숙사 식당 메뉴를 제공한다.
```
### 개발환경
```
php 5.6
에디터 atom
서버 dothome 호스팅 업체 이용
simple_html_dom 이용하여 파싱
플러스친구 API v.2 이용
```
### 부가 설명
```
dom 형식의 html 을 파싱하여 날짜+파일명.json 으로 저장
저장한 json 파일을 불러와 가공하여 사용자에게 전달
```
